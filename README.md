## How to build

1. Install dependencies: git qt-base gcc
2. git clone https://github.com/K3V3L/CourseWork_ParallelComputing
3. cd CourseWork_ParallelComputing
4. mkdir build && cd build
5. qmake ..
6. make 
7. sudo cp invertedindex /usr/bin
